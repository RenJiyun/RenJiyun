## 玩转Jenkins

### api

通过Jenkins提供的api，可以做以下一些事情：

1. 通过脚本部署和设置Jenkins，给交付团队快速搭建一套持续集成系统
2. 通过脚本生成而不是人工配置任务，使得任务生成可以被版本系统管理
3. 可以将Jenkins集成进其他效能类系统，也算是一种二次开发
4. 开发人员可以在本地通过命令的方式查看Jenkins系统的状态，比较方便

当然上面说的一些点是需要靠console script
